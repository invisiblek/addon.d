cm-wallet-addond
================

Addon.d script for Cyanogenmod to backup Google Wallet and restore it during updates.

Assumes Wallet was installed to /system/app/wallet.apk

I wrote this script to backup my Wallet installed using "Wallet Installer" for the Galaxy S3
from http://forum.xda-developers.com/showthread.php?t=2044362

I am in no way responsible for any damages to your device that using this script may cause.
